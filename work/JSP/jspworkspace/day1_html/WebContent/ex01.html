HTML

HyperText Markup Language

HyperText : 링크로 연결되는 문서
Markup : 태그를 이용하여 웹문서 형식과 내용을 지정

meta 태그, 메타데이터

브라우저가 html 문서를 해석하는데 필요한 정보. keyword 속성값은 검색엔진에 제공되는 정보이기도 하다.

html 태그

태그 이름으로 태그의 기능을 알수 있음.
태그 내부에 속성을 설정할 수 있음.

Element(요소)
- 시작태그부터 종료태그까지
- 종 료 태 그 없는 것도 있음.<br>,<hr> 태그 등
- 필요한 값 설정을 위해 태그 안에 속성도 있음.
block 요소
- 브라우저 화면 영역 전체를 너비(width) 로 한다.스타일속성으로 너비와 높이 변경 할수있다.
- 줄 바꿈이 되는 태그. p,div,h1~h6
inline 요소
- 태그 안의 내용만큼만 너비가 자동으로 설정된다.너비와 높이 변경안된다.
- 줄 바꿈 없이 내용을 담는 형식의 태그. b,span

태그 속성은 태그의 기능을 세부적으로 설정한다. 스타일을 설정할때 사용한다.
